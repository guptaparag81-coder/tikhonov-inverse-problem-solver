# Tikhonov Regularisation for Ill-Posed Inverse Problems

A numerical inverse-modelling project demonstrating Tikhonov regularisation and L-curve analysis for recovering unknown source signals from noisy measurements.

## Objective

The purpose of this project is to solve an ill-posed inverse problem where noisy sensor measurements are used to recover an unknown source profile.

The project demonstrates how regularisation techniques stabilise inverse solutions and prevent noise amplification.

## Features

- Forward model construction
- Ill-posed inverse problem formulation
- Noisy measurement generation
- Naive inversion
- Tikhonov regularisation
- L-curve analysis
- Recovery error evaluation
- Scientific visualisation
